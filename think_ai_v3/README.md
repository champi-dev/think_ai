```bash

pip install -r requirements.txt

# Run the server
python app.py

# Or with environment variables
PORT=8080 THINK_AI_COLOMBIAN=true python app.py
```

## API Endpoints

- `GET /` - System info
- `GET /health` - Health check
- `POST /api/v1/generate` - Generate text with consciousness
- `POST /api/v1/knowledge/store` - Store knowledge (O(1) access)
- `GET /api/v1/knowledge/{key}` - Retrieve knowledge
- `POST /api/v1/knowledge/query` - Query knowledge base
- `GET /api/v1/intelligence/status` - Intelligence metrics
- `POST /api/v1/meditate` - Enter meditation mode
- `GET /api/v1/config` - Get configuration

## Environment Variables

- `PORT` - Server port (default: 8080)
- `THINK_AI_MODEL` - Model to use (default: Qwen/Qwen2.5-Coder-1.5B)
- `THINK_AI_COLOMBIAN` - Enable Colombian mode (default: true)
- `THINK_AI_ENV` - Environment (development/production)
- `HF_TOKEN` - HuggingFace token for model downloads
