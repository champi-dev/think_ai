## 📊 Quick Reference

### Commands
```bash
# Installation
pip install think-ai-consciousness

# Start server
think-ai serve

# Quick chat
think-ai chat "Hello"

# Training
think-ai train --topic "subject" --file data.txt

# Help
think-ai --help
```

### Code Snippets
```python
# Basic usage
from think_ai import ThinkAI
ai = ThinkAI()
response = ai.chat("Hello!")

# Training
ai.train("topic", "knowledge")

# Plugins
ai.install_plugin("plugin-name")
```

### Configuration
```yaml
# config.yaml
model: gpt-3.5-turbo
temperature: 0.7
max_tokens: 500
cache_enabled: true
```
