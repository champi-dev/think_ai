# `think-ai-codellama`

The `think-ai-codellama` crate provides an interface to the Code Llama language model.

## Key Components

*   **`CodeLlamaClient`:** A client for the Code Llama language model.
*   **`CodeLlamaRequest`:** A request to the Code Llama language model.
*   **`CodeLlamaResponse`:** A response from the Code Llama language model.
