# `think-ai-qwen`

The `think-ai-qwen` crate provides an interface to the Qwen language model.

## Key Components

*   **`QwenClient`:** A client for the Qwen language model.
*   **`QwenRequest`:** A request to the Qwen language model.
*   **`QwenResponse`:** A response from the Qwen language model.
