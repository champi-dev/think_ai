# `think-ai-vector`

The `think-ai-vector` crate is responsible for managing the system's vector space. It includes a number of components that are used to store, retrieve, and process vectors.

## Key Components

*   **`VectorEngine`:** The main entry point for the vector crate. It provides a high-level API for interacting with the system's vector space.
*   **`VectorSearch`:** A component that is responsible for searching the vector space.
*   **`VectorStorage`:** A component that is responsible for storing vectors.

## Dependencies

The `think-ai-vector` crate depends on the following core crates:

*   `think-ai-utils`
