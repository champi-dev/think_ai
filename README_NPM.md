# Think AI - NPM Package

## Quick Start

```bash
npm install think-ai-consciousness
```

```javascript
import ThinkAI from 'think-ai-consciousness';

const ai = new ThinkAI({
  colombianMode: true,
  apiKey: process.env.ANTHROPIC_API_KEY
});

const response = await ai.think("What is consciousness?");
console.log(response);
```

## Features

- 🧠 Conscious AI responses
- 🇨🇴 Colombian expressions
- 🚀 Fast performance
